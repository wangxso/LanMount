## 📝 截图指南

### 如何截图

1. **使用 macOS 截图工具**
   ```bash
   # 截取窗口
   Cmd + Shift + 4，然后按空格键，点击窗口
   
   # 或使用截图应用
   Cmd + Shift + 5
   ```

2. **推荐设置**
   - 窗口大小：尽量使用标准尺寸
   - 背景：干净的桌面背景
   - 内容：显示有意义的数据（不要空白）
   - 格式：PNG（保持透明度）

3. **优化截图**
   ```bash
   # 调整大小（如果需要）
   sips -Z 1600 screenshot.png
   
   # 压缩（可选）
   pngquant screenshot.png --quality=80-95
   ```

### 截图内容建议

**Dashboard（主界面）：**
- 显示 2-3 个已挂载的卷
- 显示连接状态（已连接/未连接）
- 显示一些统计信息

**Network Scanner（网络扫描）：**
- 显示 3-5 个发现的服务器
- 显示不同类型的服务器（NAS、Mac、Windows）
- 显示扫描进度或完成状态

**Mount Config（挂载配置）：**
- 填写示例配置（使用示例数据）
- 显示所有可用选项
- 显示验证状态
